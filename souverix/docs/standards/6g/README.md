# 6G Standards Watchlist

This document tracks authoritative 6G standards, frameworks, and research initiatives relevant to Souverix IMS evolution.

## Watchlist

### Primary Standards and Frameworks

- **ITU IMT-2030 (6G) program page**
  - **Link**: [ITU IMT-2030 Working Track](https://www.itu.int/en/ITU-R/study-groups/rsg5/rwp5a/imt-2030/Pages/default.aspx)
  - **Description**: Working track landing page for IMT-2030 (6G) development
  - **Why Track**: Authoritative source for 6G framework and timeline

- **ITU-R Recommendation M.2160**
  - **Link**: [ITU-R M.2160 - Framework and objectives for IMT-2030](https://www.itu.int/rec/R-REC-M.2160)
  - **Description**: Framework & objectives for IMT-2030
  - **Why Track**: Defines 6G vision, use cases, and technical requirements

- **3GPP Release 20 status page**
  - **Link**: [3GPP Release 20](https://www.3gpp.org/release-20)
  - **Description**: Early 6G study work and timelines
  - **Why Track**: 3GPP's path to 6G standards, IMS evolution in 6G context

### Major Ecosystem Roadmaps and Research Flagships

- **ATIS Next G Alliance - National 6G Roadmap**
  - **Link**: [Next G Alliance](https://www.nextgalliance.org/)
  - **Description**: North America's roadmap stream
  - **Why Track**: North American perspective on 6G priorities and timelines

- **EU SNS JU (Smart Networks & Services Joint Undertaking)**
  - **Link**: [SNS JU Program](https://smart-networks.europa.eu/)
  - **Description**: Good "what's actually happening" updates
  - **Why Track**: European 6G research and deployment initiatives

- **Hexa-X-II (EU flagship 6G project)**
  - **Link**: [Hexa-X-II](https://hexa-x-ii.eu/)
  - **Description**: System blueprint thinking
  - **Why Track**: Comprehensive 6G system architecture and design principles
